# Monster Master
A somewhat interactive monster. Based around a Particle Photon Board.
It uses a PIR motion sensors, a pressure pad, a servo, a solenoid,
an Adafruit SoundFX board and some moxie to bring some excitement to your dull holiday or birthday present giving.
